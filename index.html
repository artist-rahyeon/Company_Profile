DATA

GRIP 교재 데이터 · 수학비서 문항 데이터베이스 기반

DATA PARTNER

수학비서

SUGAKBISSEO · MATH ITEM DATABASE

각인학원의 수학 교재 GRIP 시리즈는 수학비서의 방대한 문항 데이터베이스를 기반으로 설계·검증됩니다. 검증된 문항 풀과 난이도 지표를 활용해 학습 단계별 난도 곡선을 정교하게 설계하고, 학생의 학습 이력에 따라 문항 구성을 조정합니다.

데이터 파트너 수학비서 (Sugakbisseo)
데이터 성격 수학 문항 데이터베이스 · 난이도 지표 · 풀이 이력
활용 영역 GRIP 시리즈 교재 개발 · 난도 설계 · 학습 단계 구성
적용 교재 GRIP INCEPTION · PATTERN · DECODE
연계 시점 2023 ~ 현재
